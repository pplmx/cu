# Plan: Phase 7 — Device Mesh Detection

**Phase:** 7
**Plan:** 07-01
**Requirements:** MGPU-01, MGPU-02, MGPU-03, MGPU-04
**Created:** 2026-04-24

## Goal Summary

Enable GPU topology discovery and peer memory access between devices. This phase provides the foundational layer for all multi-GPU operations: device enumeration, peer capability queries, cached topology lookups, and async peer-to-peer copy primitives.

## Requirements Coverage

| Requirement | Description | Success Criteria |
|-------------|-------------|------------------|
| MGPU-01 | Device enumeration and properties query | User can enumerate all CUDA devices and query properties (memory, compute capability, clock rates) |
| MGPU-02 | Peer access capability between GPU pairs | User can query peer access capability between any GPU pair with validation (cudaDeviceCanAccessPeer) |
| MGPU-03 | Peer access matrix with cached lookup | User can access a cached peer access matrix for O(1) topology lookups |
| MGPU-04 | Async peer-to-peer copy primitives | Async peer-to-peer copy works with existing StreamManager streams on peer-capable GPUs |

## Implementation Details

### 1. DeviceMesh Singleton (MGPU-01, MGPU-02, MGPU-03)

**File:** `include/cuda/mesh/device_mesh.h`

```cpp
namespace cuda::mesh {

struct PeerInfo {
    int device_id;
    size_t global_memory_bytes;
    int compute_capability_major;
    int compute_capability_minor;
    char name[256];
};

class DeviceMesh {
public:
    static DeviceMesh& instance();

    // MGPU-01: Device enumeration
    int device_count() const { return device_count_; }
    std::vector<PeerInfo> get_mesh_devices() const;

    // MGPU-02: Peer access capability
    bool can_access_peer(int src, int dst) const;

    // MGPU-03: Cached peer access matrix
    const PeerCapabilityMap& peer_capabilities() const { return peer_capabilities_; }

    // Lazy initialization
    void initialize();

    DeviceMesh(const DeviceMesh&) = delete;
    DeviceMesh& operator=(const DeviceMesh&) = delete;

private:
    DeviceMesh();

    int device_count_ = 0;
    PeerCapabilityMap peer_capabilities_;
    bool initialized_ = false;
};

class PeerCapabilityMap {
public:
    void initialize(int device_count);
    bool can_access(int src, int dst) const;
    int device_count() const { return static_cast<int>(matrix_.size()); }

private:
    std::vector<std::vector<bool>> matrix_;  // matrix_[i][j] = can i access j
};

}  // namespace cuda::mesh
```

**Implementation:** `src/cuda/mesh/device_mesh.cu`

- Lazy initialization: query device count, then probe all pairs with `cudaDeviceCanAccessPeer()`
- Cache results in `PeerCapabilityMap` for O(1) lookups
- Handle single-GPU gracefully (matrix will have size 1x1)
- Throw on CUDA errors during initialization with clear error messages

### 3. PeerCopy Async Copy Primitive (MGPU-04)

**File:** `include/cuda/mesh/peer_copy.h`

```cpp
namespace cuda::mesh {

class PeerCopy {
public:
    PeerCopy() = default;

    // Async peer-to-peer copy
    // Throws if peer access not available between devices
    void copy_async(void* dst, const void* src, size_t bytes,
                    int dst_device, int src_device,
                    cudaStream_t stream = 0);

    // Synchronous copy (blocks until complete)
    void copy(void* dst, const void* src, size_t bytes,
              int dst_device, int src_device);

    // Check if peer access is available (does not enable it)
    bool peer_access_available(int src, int dst) const;

    // Enable peer access if not already enabled
    void enable_peer_access(int src, int dst);

    PeerCopy(const PeerCopy&) = delete;
    PeerCopy& operator=(const PeerCopy&) = delete;

private:
    std::set<std::pair<int, int>> enabled_peer_pairs_;
};

}  // namespace cuda::mesh
```

**Implementation:** `src/cuda/mesh/peer_copy.cu`

- Use `cudaMemcpyAsync` with `cudaMemcpyDeviceToDevice` for async copies
- Validate peer access with `DeviceMesh::instance().can_access_peer()` before enabling
- Track enabled peer pairs to avoid redundant `cudaEnablePeerAccess` calls
- Single-GPU fallback: if `src_device == dst_device`, use normal `cudaMemcpyAsync`

#### MGPU-01: Device Enumeration

TEST_F(DeviceMeshTest, EnumerateDevices) {
    auto& mesh = DeviceMesh::instance();
    EXPECT_GT(mesh.device_count(), 0);

TEST_F(DeviceMeshTest, GetMeshDevices) {
    auto& mesh = DeviceMesh::instance();
    auto devices = mesh.get_mesh_devices();
    EXPECT_EQ(devices.size(), mesh.device_count());

    for (const auto& info : devices) {
        EXPECT_GE(info.device_id, 0);
        EXPECT_LT(info.device_id, mesh.device_count());
        EXPECT_GT(info.global_memory_bytes, 0);
        EXPECT_GE(info.compute_capability_major, 0);
    }
}

#### MGPU-04: Async Peer Copy

```cpp
TEST_F(PeerCopyTest, SingleDeviceCopy) {
    PeerCopy copier;

    std::vector<float> h_src(1024, 1.0f);
    std::vector<float> h_dst(1024, 0.0f);

    cuda::memory::Buffer<float> d_src(1024);
    cuda::memory::Buffer<float> d_dst(1024);

    d_src.copy_from(h_src.data(), 1024);

    // Single device copy (no peer involved)
    copier.copy_async(d_dst.data(), d_src.data(), 1024 * sizeof(float),
                      0, 0);

    CUDA_CHECK(cudaStreamSynchronize(0));

    d_dst.copy_to(h_dst.data(), 1024);

    for (float val : h_dst) {
        EXPECT_EQ(val, 1.0f);
    }
}

TEST_F(PeerCopyTest, PeerAccessAvailable) {
    PeerCopy copier;
    auto& mesh = DeviceMesh::instance();

    // Self-access should always be available
    EXPECT_TRUE(copier.peer_access_available(0, 0));
}

TEST_F(PeerCopyTest, EnablePeerAccess) {
    PeerCopy copier;
    auto& mesh = DeviceMesh::instance();

    if (mesh.device_count() > 1) {
        EXPECT_NO_THROW({
            copier.enable_peer_access(0, 1);
        });
    }
}

TEST_F(PeerCopyTest, PeerCopyWithStream) {
    PeerCopy copier;
    auto& mesh = DeviceMesh::instance();

    if (mesh.device_count() < 2) {
        GTEST_SKIP() << "Requires multiple GPUs";
    }

    cuda::memory::Buffer<float> d_src(1024);
    cuda::memory::Buffer<float> d_dst(1024);

    // Fill source
    std::vector<float> h_src(1024, 42.0f);
    d_src.copy_from(h_src.data(), 1024);

    // Use stream from StreamManager
    auto stream = cuda::async::global_stream_manager().get_stream();

    if (mesh.can_access_peer(0, 1)) {
        copier.copy_async(d_dst.data(), d_src.data(), 1024 * sizeof(float),
                          1, 0, stream);
        CUDA_CHECK(cudaStreamSynchronize(stream));
    } else {
        GTEST_SKIP() << "Peer access not available between devices";
    }
}
```

TEST_F(DeviceMeshTest, SingleGpuFallback) {
    auto& mesh = DeviceMesh::instance();

    EXPECT_TRUE(mesh.can_access_peer(0, 0));

TEST_F(PeerCopyTest, SingleGpuCopy) {
    PeerCopy copier;

    // Should work on single GPU without peer access
    cuda::memory::Buffer<int> d_src(256);
    cuda::memory::Buffer<int> d_dst(256);

    std::vector<int> h_data(256, 123);
    d_src.copy_from(h_data.data(), 256);

    copier.copy_async(d_dst.data(), d_src.data(), 256 * sizeof(int), 0, 0);

    CUDA_CHECK(cudaDeviceSynchronize());

    std::vector<int> h_result(256);
    d_dst.copy_to(h_result.data(), 256);

    for (int val : h_result) {
        EXPECT_EQ(val, 123);
    }

## Success Criteria

1. **Device enumeration works:** `DeviceMesh::instance().device_count()` returns correct count
2. **Properties accessible:** `get_mesh_devices()` returns device properties for all GPUs
3. **Peer validation works:** `can_access_peer()` queries peer capability with validation
4. **Cached lookup O(1):** Multiple calls return same result without re-querying
5. **Async copy works:** `PeerCopy::copy_async()` transfers data between GPUs
6. **Stream integration:** Copies work with `StreamManager` streams
7. **Single-GPU fallback:** All operations work correctly on single-GPU systems
8. **Tests pass:** All tests pass on single-GPU CI runner

## Pitfalls Addressed

| Pitfall | Mitigation |
|---------|------------|
| Peer access without validation (PITFALL-1) | Always check `cudaDeviceCanAccessPeer()` before enabling |
| Single-GPU fallback broken (PITFALL-6) | Test on single-GPU CI, single-GPU fast path |
| Stream per device scope (PITFALL-8) | `ScopedDevice` RAII guard, peer copy with stream parameter |
| Memory consistency (PITFALL-2) | Use `cudaStreamSynchronize` after peer copies |
