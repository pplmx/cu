### Communicator Split

Use ncclCommSplit (NCCL 2.26+):

```cpp
ncclCommSplit(
    original_comm,
    color,      // Same color = same new communicator
    key,        // Ordering within communicator
    new_comm,
    0           // flags
);
```

For hybrid TP+DP:

- TP group: all ranks with same TP position share TP communicator
- DP group: all ranks with same DP position share DP communicator

### Interleaved Schedule

Alternative to 1F1B with reduced bubbles:

```text
Standard 1F1B:  F F F F B B B B
Interleaved:    F F F B F F B B (M=4, K=2)
```

Reduces bubbles by ~50% but increases activation memory.
