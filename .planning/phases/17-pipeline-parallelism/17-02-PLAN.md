### Header: include/cuda/pipeline/activation_buffer.h

```cpp
class ActivationBuffer {
public:
    ActivationBuffer(int device, size_t capacity);

    void* get_ping() { return ping_; }
    void* get_pong() { return pong_; }
    void swap() { std::swap(ping_, pong_); }
    bool is_ping_active() const { return active_is_ping_; }

private:
    void* ping_;
    void* pong_;
    bool active_is_ping_;
    size_t capacity_;
};
```

### Ping-Pong Strategy

```text
Stage N computes forward, produces activation[N]
  -> While computing backward[N-1], send activation[N] to Stage N+1
Stage N+1 receives into ping buffer
  -> Swap when ready, use new ping for computation
```

Double-buffering allows computation and communication to overlap.

### P2P Memory Copy

Use cudaMemcpyAsync with cudaMemcpyDeviceToDevice.
Verify peer access via DeviceMesh::can_access_peer() first.
