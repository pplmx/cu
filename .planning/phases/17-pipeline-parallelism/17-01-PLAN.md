### Header: include/cuda/pipeline/pipeline_scheduler.h

```cpp
enum class ScheduleType {
    OneForwardOneBackward,  // Standard 1F1B
    Interleaved             // Interleaved schedule
};

class PipelineScheduler {
public:
    PipelineScheduler(
        NcclContext& ctx,
        int num_stages,
        int num_microbatches,
        int microbatch_size);

    void forward(int stage, int microbatch, const Tensor& input, Tensor& output);
    void backward(int stage, int microbatch, const Tensor& grad_output, Tensor& grad_input);

    void run();
    void wait();

private:
    void schedule_1f1b();
    void schedule_interleaved();

    int num_stages_;
    int num_microbatches_;
    int microbatch_size_;
    ScheduleType schedule_type_;
    std::vector<std::unique_ptr<Tensor>> activations_;
    std::vector<std::unique_ptr<Tensor>> gradients_;
};
```

### 1F1B Schedule Algorithm

For K stages, M microbatches, schedule:
```
for i in range(M + K - 1):
    if i < K:        # Warmup - only forward
        forward(i, 0)
    elif i < M:      # Steady state - 1F1B
        backward(i-K, M-1)
        forward(i, i-K+1)
    else:            # Cooldown - only backward
        backward(i-K, M+K-1-i)
```

### Key Parameters

- `num_stages`: Number of pipeline stages (layers / GPUs)
- `num_microbatches`: Total micro batches to process
- `microbatch_size`: Samples per microbatch
- Optimal: M >= 4K for < 10% bubble overhead
