# Plan 15-01: NCCL AllGather Implementation

## Objective

Implement `NcclAllGather` class for gathering data from all ranks into a contiguous output buffer.

## Requirements Addressed

- EXTD-01: All-gather for row-parallel activation gathering

## Implementation

### Header: include/cuda/nccl/nccl_all_gather.h

```cpp
class NcclAllGather : public NcclCollective {
public:
    explicit NcclAllGather(NcclContext& ctx);

    NcclResult all_gather_async(
        const void* send_data,
        void* recv_data,
        size_t send_count,
        ncclDataType_t dtype,
        cudaStream_t stream);

    NcclResult all_gather(
        const void* send_data,
        void* recv_data,
        size_t send_count,
        ncclDataType_t dtype,
        cudaStream_t stream);

    static size_t required_recv_buffer_size(size_t send_count, int device_count);
};
```

### Source: src/cuda/nccl/nccl_all_gather.cpp

- Implement all_gather_async using ncclAllGather
- Implement all_gather wrapping async + stream synchronize
- Add size helper for buffer allocation

### Fallback (when NCCL unavailable)

When `!has_nccl()`, implement P2P all-gather:

- For each rank i != current: memcpy send_data to rank i's recv segment
- Use cudaMemcpyAsync with peer access enabled
- Synchronize all transfers before returning
