## Implementation

### Communicator Cache

Add to nccl_context.cpp:

```cpp
class NcclCommCache {
    static constexpr size_t MAX_CACHE_ENTRIES = 16;
    std::unordered_map<CacheKey, ncclComm_t, CacheKeyHash> cache_;
    std::list<CacheKey> lru_order_;
};
```

- Cache by (device_count, device_ids)
- LRU eviction when limit reached
- Thread-safe with mutex
