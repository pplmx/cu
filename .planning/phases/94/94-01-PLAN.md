# Phase 94: Kernel Fusion Analysis — Plan

## Tasks

### 1. Create KernelFusionAnalyzer (FUSION-01)

**Files:**
- `include/cuda/performance/fusion/kernel_fusion_analyzer.h`
- `src/cuda/performance/fusion/kernel_fusion_analyzer.cpp`

**Implementation:**
- Define `FusionOpportunity` struct with op1, op2, pattern type, location
- Define `OpType` enum: Matmul, Conv, Activation, Pooling, Reduction, ElementWise
- `KernelFusionAnalyzer::detect_opportunities()` scans operation pairs
- Pattern library for 10+ known fusions

### 2. Create FusionProfitabilityModel (FUSION-02)

**Files:**
- `include/cuda/performance/fusion/fusion_profitability.h`
- `src/cuda/performance/fusion/fusion_profitability.cpp`

**Implementation:**
- `FusionProfitabilityModel` with configurable thresholds
- Break-even calculation: `launch_overhead_saved > memory_coalescing_tradeoff`
- Configurable `launch_overhead_us` (default: 100us)
- Profitability score: 0.0 to 1.0

### 3. Create FusionRecommendationEngine (FUSION-03)

**Files:**
- `include/cuda/performance/fusion/fusion_recommendation.h`
- `src/cuda/performance/fusion/fusion_recommendation.cpp`

**Implementation:**
- `ConfidenceLevel` enum: HIGH, MEDIUM, LOW
- `FusionRecommendation` struct with:
  - opportunity details
  - confidence level
  - before/after cost estimate
  - actionable suggestions
- JSON export for CI integration

### 4. Create Fusion Pattern Library

**File:** `include/cuda/performance/fusion/fusion_patterns.h`

**Patterns:**
| Pattern | Components | Confidence |
|---------|-----------|------------|
| matmul_bias_act | matmul → bias → activation | HIGH |
| conv_bias_act | conv → bias → activation | HIGH |
| relu_pool | ReLU → max_pool | HIGH |
| elementwise_chain | add → mul → add | HIGH |
| reduction_norm | reduce → layer_norm | MEDIUM |
| memory_bound_adjacent | any two memory-bound ops | LOW |

### 5. Write Unit Tests

**File:** `tests/performance/fusion_analyzer_test.cpp`

**Test cases:**
- Detect matmul+bias+ReLU pattern
- Detect conv+bias+SiLU pattern
- Profitability model with launch overhead
- Confidence level assignment
- JSON export format
