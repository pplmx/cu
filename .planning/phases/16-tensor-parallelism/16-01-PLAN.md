### Header: include/cuda/neural/tensor_parallel_matmul.h

```cpp
namespace cuda::neural {

enum class TensorParallelStrategy {
    ColumnParallel,  // Split output dimension
    RowParallel      // Split input dimension
};

class TensorParallelMatmul {
public:
    TensorParallelMatmul(NcclContext& ctx, TensorParallelStrategy strategy);

    void matmul(
        const float* A, const float* B, float* C,
        int m, int n, int k);

    void matmul_async(
        const float* A, const float* B, float* C,
        int m, int n, int k, cudaStream_t stream);

    static size_t required_buffer_size(int m, int n, int k, int tp_degree);
    static int recommended_tp_degree();
};

}
```

### Source: src/cuda/neural/tensor_parallel_matmul.cpp

**Column-Parallel:**

- Split B along n dimension: each GPU gets n/tp_degree columns
- Each GPU computes C_part = A @ B_part
- Use NcclAllReduce to sum partial results
- Input A replicated (no communication)
- Output C identical on all GPUs after reduction

**Row-Parallel:**

- Split A along m dimension: each GPU gets m/tp_degree rows
- Each GPU computes C_part = A_part @ B
- No reduction needed (independent output rows)
- Requires input partitioning from previous layer

### Memory Requirements

- Weights: n *k* sizeof(float) / tp_degree per GPU
- Activations: m *k* sizeof(float) (replicated for column-parallel)
- Output buffer: m *n* sizeof(float) / tp_degree per GPU
- Gradient buffer (training): 2x activation size
