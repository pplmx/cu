```cpp
class ColumnParallelLayer {
    // QKV projection: splits output (hidden_dim -> 3 * hidden_dim)
    // Input: [batch, seq, hidden_dim]
    // Output: [batch, seq, 3 * hidden_dim / tp_degree]
    TensorParallelMatmul& weight_q, weight_k, weight_v;

    void forward(const float* input, float* output, int batch, int seq, int hidden);
    void backward(/*...*/);
};

class RowParallelLayer {
    // Output projection: after ColumnParallelLayer
    // Input: [batch, seq, hidden_dim / tp_degree]
    // Output: [batch, seq, hidden_dim]
    // Uses row-parallel matmul, then all-reduce

    void forward(const float* input, float* output, int batch, int seq, int hidden);
    void backward(/*...*/);
};
```

### Key Patterns

**QKV Projection (ColumnParallelLayer):**
- Three weight matrices: W_q, W_k, W_v
- Each split along output dimension
- Each produces 1/3 of total Q/K/V output
- Concatenate Q/K/V parts on each GPU (need all-reduce)

**Output Projection (RowParallelLayer):**
- Input comes from ColumnParallelLayer output
- Weight split along input dimension
- Each GPU produces partial output
- All-reduce to reconstruct full output
