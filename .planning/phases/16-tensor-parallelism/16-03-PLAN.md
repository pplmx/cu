## Implementation

### Memory Profiler

Create include/cuda/neural/tensor_parallel_profile.h:

```cpp
struct TensorParallelProfile {
    size_t weight_shard_bytes;
    size_t activation_bytes;
    size_t gradient_bytes;
    size_t total_bytes;
    int max_tp_degree;
};

class TensorParallelProfiler {
public:
    static TensorParallelProfile profile(int batch, int seq, int hidden_dim, int tp_degree);
    static int recommend_tp_degree(int batch, int seq, int hidden_dim);
    static size_t available_memory(int device);
    static size_t peak_memory_usage();
};
```

### Memory Calculation

For hidden_dim H, batch B, seq S, TP degree K:

- Weight shard: (H *3H / K)* 4 bytes *3 (QKV) = 36* H^2 / K bytes
- Activation: (B *S* H) * 4 bytes = 4BSH bytes (column-parallel)
- Gradient: 2x activation size for optimizer state
