## Test Matrix for 3x3 Dense

```
| 1  0  2 |
| 0  3  0 |
| 4  0  5 |
```

CSR representation:
- values: [1, 2, 3, 4, 5]
- row_offsets: [0, 2, 3, 5]
- col_indices: [0, 2, 1, 0, 2]

For this matrix: max_nnz_per_row = 2

## ELL Layout for 3x3 Matrix

- values_: [1, 2, 3, 0, 4, 5] (2 elements per row)
- col_indices_: [0, 2, 1, -1, 0, 2]
- row_offsets_: [0, 2, 4, 6]

## SELL-C-2 Layout for 3x3 Matrix

With slice_height = 2:
- Slice 0 (rows 0-1): max_nnz = 2, padded elements = 4
- Slice 1 (row 2): max_nnz = 2, padded elements = 4
- slice_ptr_: [0, 4, 8]
- values_: [1, 2, 3, 0, 4, 5, 0, 0]
- col_indices_: [0, 2, 1, -1, 0, 2, -1, -1]
