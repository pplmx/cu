## ELL Format Layout

For a matrix with max_nnz_per_row = M:
- values_: row 0 values (M slots), row 1 values (M slots), ... (padded with zeros)
- col_indices_: row 0 column indices (M slots), row 1 column indices (M slots), ...
- row_offsets_: [0, M, 2M, 3M, ...] — offset to start of each row in values

## ELL Conversion Algorithm (FromCSR)

1. Calculate max_nnz_per_row from CSR (maximum nnz in any row)
2. Allocate values_ and col_indices_ with size = num_rows * max_nnz_per_row
3. For each row i in CSR:
   - Copy nnz values from CSR row i to ELL values[i * max_nnz_per_row ...]
   - Copy corresponding column indices
   - Fill remaining slots with zeros / -1 for col_indices

## SELL Format Layout

For slice_height = C:
- Rows are grouped into slices of C rows (last slice may be smaller)
- Each slice is padded independently to its maximum nnz
- slice_ptr_: offsets for each slice (including end)
- values_: all slice values concatenated with padding
- col_indices_: all slice column indices concatenated with padding

## SELL Conversion Algorithm (FromCSR)

1. Calculate number of slices: num_slices = ceil(num_rows / slice_height)
2. For each slice s (rows s*C to min((s+1)*C, num_rows)):
   - Find max_nnz for this slice
   - For each row in slice: copy values and indices
3. slice_ptr_[s] points to start of slice s in values array

From existing SparseMatrixCSC::FromCSR:

Key pattern: Two-pass algorithm - first count, then allocate and copy.

   ```cpp
   SparseMatrixELL<T> SparseMatrixELL<T>::FromCSR(const SparseMatrixCSR<T>& csr) {
       int num_rows = csr.num_rows();
       int num_cols = csr.num_cols();
       
       // Pass 1: Find max nnz per row
       int max_nnz = 0;
       for (int i = 0; i < num_rows; ++i) {
           int row_nnz = csr.row_offsets()[i + 1] - csr.row_offsets()[i];
           max_nnz = std::max(max_nnz, row_nnz);
       }

       std::vector<T> values(num_rows * max_nnz, T{0});
       std::vector<int> col_indices(num_rows * max_nnz, -1);

       for (int i = 0; i < num_rows; ++i) {
           int csr_start = csr.row_offsets()[i];
           int csr_end = csr.row_offsets()[i + 1];
           int ell_base = i * max_nnz;
           
           for (int j = csr_start; j < csr_end; ++j) {
               values[ell_base + (j - csr_start)] = csr.values()[j];
               col_indices[ell_base + (j - csr_start)] = csr.col_indices()[j];
           }
       }
       
       return SparseMatrixELL<T>(std::move(values), std::move(col_indices),
                                  num_rows, num_cols, max_nnz);
   }
   ```

   ```cpp
   template<typename T>
   SparseMatrixSELL<T> SparseMatrixSELL<T>::FromCSR(const SparseMatrixCSR<T>& csr, int slice_height) {
       int num_rows = csr.num_rows();
       int num_cols = csr.num_cols();
       
       // Calculate number of slices
       int num_slices = (num_rows + slice_height - 1) / slice_height;
       
       // Pass 1: Calculate max nnz per slice and per-row positions
       std::vector<int> slice_max_nnz(num_slices, 0);
       for (int i = 0; i < num_rows; ++i) {
           int slice_idx = i / slice_height;
           int row_nnz = csr.row_offsets()[i + 1] - csr.row_offsets()[i];
           slice_max_nnz[slice_idx] = std::max(slice_max_nnz[slice_idx], row_nnz);
       }
       
       // Calculate slice offsets
       std::vector<int> slice_ptr(num_slices + 1, 0);
       for (int s = 0; s < num_slices; ++s) {
           slice_ptr[s + 1] = slice_ptr[s] + slice_max_nnz[s] * slice_height;
       }
       int total_padded_nnz = slice_ptr[num_slices];
       
       // Allocate padded storage
       std::vector<T> values(total_padded_nnz, T{0});
       std::vector<int> col_indices(total_padded_nnz, -1);
       
       // Pass 2: Copy values and indices slice by slice
       for (int slice_idx = 0; slice_idx < num_slices; ++slice_idx) {
           int slice_start_row = slice_idx * slice_height;
           int slice_end_row = std::min(slice_start_row + slice_height, num_rows);
           int local_base = slice_ptr[slice_idx];
           
           for (int i = slice_start_row; i < slice_end_row; ++i) {
               int csr_start = csr.row_offsets()[i];
               int csr_end = csr.row_offsets()[i + 1];
               int local_row = i - slice_start_row;  // Position within slice
               
               for (int j = csr_start; j < csr_end; ++j) {
                   int ell_idx = local_base + local_row * slice_max_nnz[slice_idx] + (j - csr_start);
                   values[ell_idx] = csr.values()[j];
                   col_indices[ell_idx] = csr.col_indices()[j];
               }
           }
       }
       
       return SparseMatrixSELL<T>(std::move(values), std::move(col_indices),
                                   std::move(slice_ptr), num_rows, num_cols, slice_height);
   }
   ```
