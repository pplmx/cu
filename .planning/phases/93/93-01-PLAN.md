### 1. Create NVBlox metrics header (NVBlox-01)

**File:** `include/cuda/performance/nvblox_metrics.h`

**Implementation:**

- Define `KernelMetrics` struct with name, latency, throughput, occupancy, AI, BW, timestamp
- Define `NVBloxMetricsCollector` class with RAII initialization
- Implement `register_metric()`, `add_sample()`, `record_kernel()`

- Thread-safe via mutex protection

**Files:** `include/cuda/performance/kernel_profiler.h`

**Implementation:**

- `KernelProfiler` singleton class
- `ScopedProfiler` RAII guard for auto-start/stop
- Per-kernel hooks: `on_kernel_start()`, `on_kernel_end()`
- CUDA event-based timing (fallback if NVBlox unavailable)
- Track latency_ns, block/grid config, occupancy estimate

**Files:** `include/cuda/performance/metric_aggregators.h`

**Implementation:**

- `ArithmeticIntensityCalculator` (FLOPs / bytes accessed)

- `BandwidthAggregator` (H2D/D2H/D2D breakdown)
- Thread-safe accumulation in `NVBloxMetricsCollector`
